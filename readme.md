# Comparative methods on reticulate phylogenies

This repository contains materials for a tutorial.

- Please view the tutorial website [here](https://juliaphylo.github.io/networkPCM-tutorial/)
- This tutorial uses methods implemented in
  [PhyloNetworks](https://github.com/juliaphylo/PhyloNetworks.jl),
  [PhyloTraits](https://github.com/juliaphylo/PhyloTraits.jl),
  and visualized with
  [PhyloPlots](https://github.com/juliaphylo/PhyloPlots.jl).  
  It was tested with julia v1.11, PhyloNetworks v1.1.0,
  PhyloTraits v1.1.0 and PhyloPlots v2.1.0.

The website was rendered with
[quarto](https://quarto.org/docs/websites/) v1.7.33,
and published with quarto & github pages.

```shell
quarto render
quarto publish gh-pages
```
